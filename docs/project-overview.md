# Project Overview

## Introduction

This project demonstrates how to provision Microsoft Azure infrastructure using Terraform by following Infrastructure as Code (IaC) principles.

Instead of manually creating Azure resources through the Azure Portal, Terraform automates the deployment process using reusable configuration files.

This project is designed to improve cloud automation skills and demonstrate best practices in Azure infrastructure deployment.

---

## Business Scenario

A company wants to deploy its cloud infrastructure in a consistent and repeatable manner.

Instead of manually creating every resource, the infrastructure is defined in Terraform code so it can be deployed multiple times with minimal effort.
