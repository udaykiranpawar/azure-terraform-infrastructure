## 📌 Project Overview

This project demonstrates how to provision Microsoft Azure infrastructure using **Terraform** by following **Infrastructure as Code (IaC)** principles.

The goal of this project is to automate Azure infrastructure deployment using reusable Terraform configuration files instead of manually creating resources through the Azure Portal.

---

## 🎯 Project Objectives

- Learn Infrastructure as Code (IaC)
- Automate Azure resource deployment
- Build reusable Terraform configurations
- Understand Azure networking components
- Prepare a production-style cloud project for a portfolio

---

## ☁️ Azure Resources

This project provisions:

- Resource Group
- Virtual Network (VNet)
- Subnet
- Network Security Group (NSG)
- Storage Account
- Linux Virtual Machine
